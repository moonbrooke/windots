# AHK Scripts

A collection of my AutoHotKey scripts

### keybinds

My custom Windows keybinds.

- `Windows + Enter`: Launch Windows Terminal

### zoomy_mouse

A zoom/magnifier tool.

- `F1`: Toggle zoom
- `Shift+F1`: Reset zoom
- `Alt+F1`: Change zoom shape (circle/square)
- `Shift+Scroll`: Zoom in or out
- `Ctrl+Shift+Scroll`: Resize zoom area
